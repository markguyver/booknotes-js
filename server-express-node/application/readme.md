# Booknotes (JS): Server-Side Application

## Docker Image Environment Variables

*   `APP_PORT` (Default: `3000`)
*   `LOG_LEVEL` (Default: `error`)
*   `DB_HOSTNAME`
*   `DB_PORT` (Default: `3306`)
*   `DB_USERNAME` (Default: `root`)
*   `DB_PASSWORD`
*   `DB_SCHEMA` (Default: `booknotes`)

*   `compile`
*   `db:migrate`
*   `db:rollback`
*   `develop`
*   `documentation`
*   `minimize`
*   `start`
